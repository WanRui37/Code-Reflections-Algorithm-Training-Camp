# 2026-04-24 最大子数组和 | 合并区间 | 轮转数组 | 除了自身以外数组的乘积 | 缺失的第一个正数

## 1. 快速总结

### 1) 最大子数组和
- → `动态规划` / `贪心`
- → 连续子数组最大和
- → 前面和为负就丢掉

### 2) 合并区间
- → `排序` + `遍历`
- → 区间重叠合并
- → 先按左端点排序

### 3) 轮转数组
- → `三次反转`
- → 整体移动位置
- → 整体反，前段反，后段反

### 4) 除了自身以外数组的乘积
- → `前缀乘积` + `后缀乘积`
- → 左边乘积 * 右边乘积
- → 不用除法

### 5) 缺失的第一个正数
- → `原地哈希`
- → 数字放回对应下标
- → 数字 `x` 放到 `x-1`

## 2. 题目笔记

### 1) 最大子数组和
1. **状态定义**：`dp[i]` = 以 `nums[i]` 结尾的最大连续子数组和。
2. **决策逻辑**：对于当前位置 `nums[i]`，只有两种选择：
    - 选择 1：接在前面的子数组后面 -> `dp[i - 1] + nums[i]`
    - 选择 2：不要前面的，从 `nums[i]` 重新开始 -> `nums[i]`
