# 2026-04-24 和为K的子数组 | 滑动窗口最大值 | 最小覆盖子串

## 1. 题目笔记

### 核心思想
- **子串 / 子数组问题**：本质是**连续区间问题**。

### 常见技巧与套路
1. **求区间和、统计和为 K**
    - 方法：`前缀和` + `哈希表`
2. **求最长/最短满足条件的连续区间**
    - 方法：`滑动窗口`
3. **求窗口内最大值/最小值**
    - 方法：`单调队列`
4. **求字符出现次数、包含关系、重复情况**
    - 方法：`哈希表`
5. **固定窗口长度**
    - 方法：`滑动窗口` / `单调队列`
6. **窗口长度不固定**
    - 方法：`滑动窗口` + `哈希表`
