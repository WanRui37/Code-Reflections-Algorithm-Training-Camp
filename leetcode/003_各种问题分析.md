# 各种问题分析

## 1. 核心算法分析

### 1) 哈希
1. **一般用在**：快速查找、去重、统计次数、判断某个值是否出现过。
2. **常见关键词**：
    - `出现次数`、`是否存在`、`两数之和`、`字母异位词`、`重复元素`、`前缀和计数`
3. **典型题型**：
    - `两数之和`、`有效的字母异位词`、`最长连续序列`、`和为 k 的子数组`

### 2) 双指针
1. **一般用在**：数组、字符串、链表中，用两个位置一起移动来减少枚举。
2. **常见关键词**：
    - `有序数组`、`首尾夹逼`、`原地修改`、`反转`、`删除重复元素`、`快慢指针`
3. **典型题型**：
    - `两数之和 II`、`三数之和`、`反转字符串`、`移除元素`、`判断环形链表`

### 3) 滑动窗口
1. **一般用在**：连续子数组/子串，并且窗口可以通过左右边界移动维护状态。
2. **常见关键词**：
    - `最长/最短子串`、`连续子数组`、`满足某条件`、`窗口内最多/至少/不超过`
3. **典型题型**：
    - `无重复字符的最长子串`、`长度最小的子数组`、`找到字符串中所有字母异位词`

### 4) 前缀和
1. **一般用在**：连续区间求和/统计的问题上。
2. **常见关键词**：
    - `区间和`、`子数组和`、`和为 k`、`二维矩阵区域和`、`多次查询区间和`
3. **典型题型**：
    - `区域和检索`、`和为 k 的子数组`、`二维区域和检索`

### 5) 链表
1. **一般用在**：节点之间通过指针连接，重点是指针修改和关系维护。
2. **常见关键词**：
    - `反转链表`、`删除节点`、`合并链表`、`链表相交`、`链表成环`、`倒数第 k 个节点`
3. **典型题型**：
    - `反转链表`、`合并两个有序链表`、`删除链表倒数第 N 个节点`、`环形链表`

### 6) 二叉树
1. **一般用在**：具有左右子节点的层级结构，通常用递归、DFS、BFS。
2. **常见关键词**：
    - `前序/中序/后序`、`层序遍历`、`最大深度`、`路径和`、`最近公共祖先`、`二叉搜索树`
3. **典型题型**：
    - `二叉树最大深度`、`对称二叉树`、`路径总和`、`二叉树层序遍历`、`验证二叉搜索树`

### 7) 回溯
1. **一般用在**：枚举所有可能结果，尝试一个选择，不行就撤销。
2. **常见关键词**：
    - `所有组合`、`所有排列`、`所有子集`、`切割方案`、`棋盘问题`、`路径搜索`
3. **典型题型**：
    - `组合`、`全排列`、`子集`、`电话号码的字母组合`、`分割回文串`、`N 皇后`

### 8) 二分查找
1. **一般用在**：有序数组，或者答案具有单调性的问题。
2. **常见关键词**：
    - `有序`、`查找目标`、`最左位置`、`最右位置`、`最小满足`、`最大满足`、`答案范围`
3. **典型题型**：
    - `二分查找`、`搜索插入位置`、`在排序数组中查找元素第一个和最后一个位置`、`寻找峰值`

### 9) 堆
1. **一般用在**：动态维护最大值/最小值，或者求 Top K。
2. **常见关键词**：
    - `最大/最小`、`第 k 大`、`前 k 个`、`高频元素`、`数据流中位数`、`合并多个有序序列`
3. **典型题型**：
    - `数组中的第 K 个最大元素`、`前 K 个高频元素`、`合并 K 个升序链表`、`数据流的中位数`

### 10) 栈
1. **一般用在**：后进先出、括号匹配、表达式计算、单调栈。
2. **常见关键词**：
    - `括号匹配`、`最近一个更大/更小元素`、`温度等待天数`、`路径简化`、`字符串消除`
3. **典型题型**：
    - `有效的括号`、`每日温度`、`下一个更大元素`、`最小栈`、`逆波兰表达式求值`

### 11) 贪心算法
1. **一般用在**：每一步都选当前最优，并且局部最优可以推出全局最优。
2. **常见关键词**：
    - `最多/最少`、`区间选择`、`分发`、`跳跃`、`买卖股票`、`排序后选择`
3. **典型题型**：
    - `分发饼干`、`跳跃游戏`、`加油站`、`无重叠区间`、`买卖股票的最佳时机 II`

### 12) 动态规划
1. **一般用在**：问题可以拆成子问题，并且子问题会重复出现。
2. **常见关键词**：
    - `最大/最小方案`、`多少种方法`、`能不能达到`、`选择或不选择`、`背包`、`路径`、`子序列`
3. **典型题型**：
    - `爬楼梯`、`不同路径`、`最长递增子序列`、`最长公共子序列`、`背包问题`、`打家劫舍`

---

## 2. 核心总结表

| 算法类型 | 核心应用场景 | 关键点 |
| :--- | :--- | :--- |
| **哈希** | 查找和计数 | 空间换时间，快速索引 |
| **双指针** | 两个位置一起移动 | 有序性，降低维度 |
| **滑动窗口** | 连续区间动态伸缩 | 维护区间合法性 |
| **前缀和** | 连续区间求和 | 预处理，快速查询区间 |
| **链表** | 修改指针关系 | 边界处理，指针断开重连 |
| **二叉树** | 递归与遍历 | 结构化特征，DFS/BFS |
| **回溯** | 枚举所有可能 | 尝试与撤销，状态重置 |
| **二分查找** | 有序或单调性查找 | 边界确定，折半缩小范围 |
| **堆** | 动态 Top K | 维护极值，数据流处理 |
| **栈** | 后进先出与单调关系 | 匹配与回溯状态 |
| **贪心** | 局部最优推全局 | 无后效性，当前最优选择 |
| **动态规划** | 子问题递推 | 状态转移，重复子问题 |
