### Python 里面数据结构用法

#### 1) 列表（List）
- **增**：`append(x)` (末尾添加), `insert(index, x)` (指定位置插入), `extend(iterable)` (合并)
- **删**：`remove(x)` (删指定值), `pop([index])` (删索引处并返回), `del list[index]`
- **改**：`list[index] = new_value`
- **查**：`list[index]`, `index(x)`, `count(x)`, `in` (判断是否存在)

#### 2) 元组（Tuple）
- **特点**：不可变（Immutable），一旦创建无法增删改。
- **查**：`tuple[index]`, `index(x)`, `count(x)`, `in`

#### 3) 字典（Dict）
- **增/改**：`dict[key] = value`, `update({key: value})`
- **删**：`pop(key)`, `del dict[key]`, `popitem()` (删最后一个键值对)
- **查**：`dict[key]`, `get(key, default)`, `keys()`, `values()`, `items()`, `in`

#### 4) 集合（Set）
- **增**：`add(x)`, `update(iterable)`
- **删**：`remove(x)`, `discard(x)` (删除不存在的值不报错), `pop()` (随机删)
- **查**：`in` (由于无序，不支持索引，查找效率极高)

#### 总结表

| 数据结构 | 关键字 | 增 | 删 | 改 | 查 | 特点 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **列表** | `list` | `append` / `insert` | `remove` / `pop` | `list[i] = x` | `list[i]` / `in` | 有序、可变 |
| **元组** | `tuple` | - | - | - | `t[i]` / `in` | 有序、不可变 |
| **字典** | `dict` | `d[k] = v` | `pop` / `del` | `d[k] = v` | `d[k]` / `get` | 键值对、键唯一 |
| **集合** | `set` | `add` / `update` | `remove` / `pop` | - | `in` | 无序、去重 |
