# 格式化规范

```markdown
    修改好格式，按照你理解的层级去给标号或者缩进。

    ## 缩进规范
    - **统一使用 4 个空格**作为缩进层级。

    ## 层级标号规范
    根据内容层级，依次使用：
    1. `###` 大章节（如：题目笔记、核心思想等）
    2. `### 1)` 题目编号（如：`### 1) 两数之和`）
    3. `#### 推荐思路` 推荐解法
    4. `1.` `2.` `3.` 内容编号
    5. `-` 子列表项（配合 4 空格缩进）

    ## 注意事项
    - 不增改内容，只完善格式。
    - 代码相关内容使用反引号标注。
```
