# 动态规划总结：从二维到一维的压缩套路

## 1. 核心结论
- **二维压一维的本质**：发现 `dp[i][j]` 这一层只依赖“上一层”和“当前层左边”，因此可以使用一维数组边更新边复用。

---

## 2. 从二维公式出发
1. **状态定义**：`dp[i][j]` 表示前 `i+1` 个物品在容量/金额为 `j` 时的最优值。
2. **转移方程**（以完全背包为例）：
    - `dp[i][j] = min(dp[i-1][j], dp[i][j-w] + 1)`
    - 其中 `dp[i-1][j]` 代表不用当前物品（依赖上一行）。
    - 其中 `dp[i][j-w] + 1` 代表使用当前物品一次（依赖当前行左侧）。

---

## 3. 为什么能压成一维
1. **空间复用**：计算第 `i` 行时只需上一行同列和当前行左边的数据，没必要存储整张表。
2. **状态覆盖**：让一维数组 `dp[j]` 在遍历过程中承担双重角色：
    - 更新前：表示上一行结果 `dp[i-1][j]`。
    - 更新后：表示当前行结果 `dp[i][j]`。
3. **公式简化**：
    - 二维：`dp[i][j] = min(dp[i-1][j], dp[i][j-w] + 1)`
    - 一维：`dp[j] = min(dp[j], dp[j-w] + 1)`

---

## 4. 遍历顺序（压缩的核心）
1. **0/1 背包：容量倒序**
    - **理由**：每个物品只能用一次，倒序遍历确保当前轮更新（当前行）不会覆盖掉后续计算所需的上一行数据。
    - **模板**：
    ```python
    for i in range(n):
        for j in range(bag, weight[i] - 1, -1):
            dp[j] = ...
    ```
2. **完全背包：容量正序**
    - **理由**：每个物品可以重复使用，正序遍历允许当前轮更新过的结果（当前行左侧）继续参与当前行的计算。
    - **模板**：
    ```python
    for i in range(n):
        for j in range(weight[i], bag + 1):
            dp[j] = ...
    ```

---

## 5. 通用模板
1. **0/1 背包二维压一维**
    - 二维：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-w] + v)`
    - 一维：
    ```python
    for i in range(n):
        for j in range(bag, weight[i] - 1, -1):
            dp[j] = max(dp[j], dp[j - weight[i]] + value[i])
    ```
2. **完全背包二维压一维**
    - 二维：`dp[i][j] = max(dp[i-1][j], dp[i][j-w] + v)`
    - 一维：
    ```python
    for i in range(n):
        for j in range(weight[i], bag + 1):
            dp[j] = max(dp[j], dp[j - weight[i]] + value[i])
    ```

---

## 6. 运算符对照
1. **最大价值**：`dp[j] = max(dp[j], dp[j-w] + v)`
2. **最少个数**：`dp[j] = min(dp[j], dp[j-w] + 1)`
3. **方案数**：`dp[j] += dp[j-w]`
- **框架核心**：当前状态 = 不选当前物品的结果 vs 选当前物品后的结果。

---

## 7. 实战举例：完全平方数
1. **场景判断**：平方数可以重复取，属于完全背包。
2. **代码实现**：
```python
import math

class Solution:
    def numSquares(self, n: int) -> int:
        nums = [i * i for i in range(1, int(math.sqrt(n)) + 1)]
        dp = [float('inf')] * (n + 1)
        dp[0] = 0

        for num in nums:
            # 完全背包，正序遍历
            for j in range(num, n + 1):
                dp[j] = min(dp[j], dp[j - num] + 1)

        return dp[n]
```

---

## 8. 快速判断技巧
1. **看二维转移公式的第二项依赖谁**：
    - 若依赖 `dp[i-1][j-w]`：上一行数据 -> **0/1 背包** -> 压缩后 **倒序**。
    - 若依赖 `dp[i][j-w]`：当前行左侧 -> **完全背包** -> 压缩后 **正序**。

---

## 9. 压缩思维口诀
> **看二维第二项，**
> **若是上一行就倒序，**
> **若是当前行就正序。**

---

## 10. 总结
- **本质**：用一个数组滚动覆盖每一行结果。
- **关键**：通过观察二维转移第二项依赖的是“上一行”还是“当前行”，决定一维数组的遍历方向。
